Quick Start

Docker Compose

git clone https://github.com/NickCrew/apparatus.git
cd apparatus
docker compose --profile chimera up -d

From Source

git clone https://github.com/NickCrew/apparatus.git
cd apparatus
pnpm install && pnpm build
pnpm dev:server

CLI

npm install -g @atlascrew/apparatus-cli
apparatus --url http://localhost:8090 status
